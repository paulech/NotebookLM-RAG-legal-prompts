# Auditor RAG Jurídico - NotebookLM Prompts

Repositorio de instrucciones de sistema optimizadas para mitigar la obsolescencia documental en arquitecturas RAG y NotebookLM tras las reformas del DNU 70/2023 y Ley 27.742 en Argentina.

## 🚀 Arquitectura del Corpus
Para garantizar la Supremacía Hermenéutica, cargue en su libreta:
1. Corpus Doctrinario: Manuales, tratados y jurisprudencia histórica.
2. Corpus Normativo Vigente: Texto actualizado de Leyes 27.742, 27.743 y decretos reglamentarios.

## 🛠️ Modos de Uso
Copie el contenido de la especialidad requerida (`/prompts`) en las "Instrucciones de la Guía" de NotebookLM.

## ⚠️ Mitigación de Alucinaciones
El sistema responderá estrictamente bajo la estructura:
- I. Objeto de Análisis
- II. Desarrollo y Contraste (con alertas normativas integradas)
- III. Conclusión y Estado de Vigencia
Si el dato no existe en las fuentes, devolverá de forma invariable el token `[Vacío Documental]`.
